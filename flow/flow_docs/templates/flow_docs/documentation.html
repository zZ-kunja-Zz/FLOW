Flow is a website built for writing. Normal writing is slow and inefficient to edit quickly to change this flow is line by line. To add formatting there are commands such as "title" and "endtitle".

  • Titles
  • Unicode Characters
  • Paragraphs

'title' and 'endtitle'

To create a title use the title keyword followed by the title you want followed by endtitle.
title
Example Title
endtitle
Note that title and endtitle must have their own lines.

Adding Unicode Characters

To add special characters use the HTML 5 syntax: &#YOUR_UNICODE;
&#8484;
